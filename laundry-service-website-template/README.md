# 54 Digital – Página institucional

Página web institucional para la empresa 54 Digital, orientada a ofrecer
soluciones tecnológicas y rastreo satelital de flotas vehiculares.

## Tecnologías utilizadas
- HTML5
- CSS3 (Flexbox)
- JavaScript

## Descripción
El proyecto se desarrolló a partir de una plantilla HTML base, la cual fue
adaptada completamente al contenido y necesidades de la empresa:
- modificación de textos y secciones
- ajustes de estilos y colores
- adaptación responsive
- agregado de funcionalidades con JavaScript

## Aprendizajes
- estructura semántica de HTML
- maquetado con CSS
- lectura y adaptación de código existente
- manipulación básica del DOM
